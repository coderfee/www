# 实现方案：朋友名片弹窗 (Friend Business Card)

**日期**: 2026-02-02
**状态**: ✅ 已完成
**目标**: 为友情链接增加“名片式”中转弹窗，提升社交仪式感，并保持全页交互体验的高度统一。

## 1. 实现概述

本方案已全面实施。友情链接点击不再直接跳转，而是通过一个精美的“名片弹窗”进行中转，展示朋友的完整简介和 URL 预览。

## 2. 核心功能

### 2.1 物理级形变动画
使用 Framer Motion 的 `layoutId` 机制，实现了从列表项到名片容器的平滑过渡。动画焦点位于朋友头像，产生了一种“头像放大展开成名片”的视觉错觉。

### 2.2 名片 UI 设计
- **大尺寸头像**: 朋友头像放大显示，增强视觉辨识度。
- **信息展示**: 包含朋友姓名、完整简介及原始 URL 预览。
- **双重交互**:
    - **一键复制**: 按钮右侧带有零抖动的图标切换反馈（Check 图标）。
    - **开启传送门**: 使用醒目的深色/反色按钮引导用户跳转。

## 3. 统一规范集成

- **触感反馈**: 触发弹窗和点击按钮均伴随 `light` 模式微振。
- **紧凑布局**: 遵循 `p-8` 等宽内边距和 `py-3` 按钮高度规范。
- **关闭逻辑**: 关闭按钮悬浮于容器下方 `-bottom-16` 位置，彻底规避误触。

## 4. 收益总结
该功能的上线标志着 `About` 页面所有外部链接（个人社交 + 友情链接）逻辑的完全闭环。不仅提升了页面的交互趣味性，也表达了博主对每一位志同道合朋友的重视。
