# 实现方案：社交链接系统化增强 (GitHub, Email, RSS, Sitemap)

**日期**: 2026-02-02
**状态**: ✅ 已完成
**目标**: 为 `AboutProfile` 组件打造一套统一的、具有丝滑动画和亲切文案的弹窗增强体系。

## 1. 最终实现概述

本方案将原本单一的外部跳转升级为深度交互的弹窗矩阵。通过统一的技术规范和 UI 语言，提升了博客的专业感与人情味。

## 2. 核心功能分解

### 2.1 GitHub：数据驱动的资历名片
- **核心数据**: 抓取 Repos、Stars、Followers 和码龄。
- **性能**: 实施 24 小时 `localStorage` 缓存策略，保护 API 配额。
- **交互**: 点击触发 `github-modal`，底部提供“去主页逛逛”按钮。

### 2.2 Email：优雅的一键写信
- **核心功能**: 提供两端对齐的复制按钮 + 立即发邮件的主动作按钮。
- **反馈**: 零布局抖动的 Check 图标切换反馈。
- **文案**: “给我写信”，“打个招呼，或者单纯聊聊想法”。

### 2.3 RSS：订阅从未如此简单
- **核心功能**: 展示 Feed 地址一键复制 + 主流阅读器推荐（NetNewsWire, Reeder 等）。
- **逃生舱**: 提供“直接查看 XML”按钮。
- **文案**: “订阅动态”，“在阅读器里第一时间看到我的新文章”。

### 2.4 Sitemap：内容博物馆目录
- **核心功能**: 展示文章总数统计、创作历程年份以及核心分类标签。
- **交互**: 引导用户查看完整的站点地图索引。

## 3. 统一的技术与设计规范

### 3.1 物理级形变动画
全量使用 Framer Motion 的 `layoutId` 机制，实现了从列表卡片到悬浮弹窗的 **Shared Element Transition**。配合统一的 `spring` 过渡参数，确保了物理反馈的真实感。

### 3.2 极致紧凑的 UI 语言
- **内边距**: 统一为 `p-6 pt-8`。
- **栅格**: 统一使用 `space-y-6` 间距。
- **按钮**: 统一高度为 `py-3`，圆角 `rounded-2xl`。
- **对齐**: 复制类按钮统一使用 `justify-between`。

### 3.3 无障碍与健壮性
- **A11y**: 完整支持 `role="dialog"` 和 `aria-label`。
- **锁定**: 使用 `useBodyScrollLock` Hook 防止弹窗时背景滚动。

## 4. 总结
整套系统不仅解决了技术上的痛点（如 `mailto:` 唤起不便、RSS 源码惊吓），更通过统一的视觉和丝滑的动画，确立了博客独特的品牌气质。
