# Newsletter 详情页杂志风重构方案

## 概述
为了保持周刊系列在视觉语言上的连贯性，我们将详情页从传统的“博客模式”重构为“沉浸式编辑模式（Editorial Mode）”。设计核心是延续列表页的“书脊”感与“画廊式”排版。

## 核心设计理念

### 1. 延续性书脊 (The Sidebar Spine)
*   **设计**：在页面左侧保留垂直的品牌/元数据区域。
*   **内容**：
    *   **垂直刊号**：巨大的、半透明的衬线体刊号作为背景装饰。
    *   **返回导航**：集成在侧边栏顶部的圆形返回按钮，支持物理引擎动效。
*   **交互**：采用 `sticky` 定位与 `transform-gpu` 加速，确保滚动时的丝滑体验。

### 2. 沉浸式刊头 (Editorial Hero Section)
*   **排版**：采用 `font-noto` 7xl 超大标题，配合 `tracking-tight` 展现报刊张力。
*   **画廊封面**：封面图采用 `21:9` 超宽比例，辅以 iOS 弥散阴影与 `3.5rem` 超大圆角，营造精美的摄影画廊既视感。

### 3. 非对称平衡布局 (Asymmetric Tri-column)
*   **结构**：采用了“左书脊 + 中正文 + 右导读”的三栏错落布局。
*   **右侧侧边栏**：
    *   **Editorial Lead**：将引言外挂至右侧，以引号和斜体形式呈现，填补视觉空白。
    *   **装饰性 Issue**：在侧边栏再次强调期数，建立多维度的品牌触点。
*   **响应式**：在 `lg` 断点下自动塌陷为单栏流式布局，保证移动端的易读性。

### 4. 动画与质感 (Motion & Texture)
*   **动画解耦**：彻底解决了 Framer Motion 与 CSS Transition 冲突导致的位移抖动和“直角变圆角”问题。
*   **玻璃拟态**：全站统一的超大模糊半径阴影，增强了页面的通透感。
