# Code of Conduct

## Our Pledge

We want this project to be a welcoming space for discussion, bug reports, and contributions. Everyone participating in this project is expected to communicate respectfully and constructively.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect differing viewpoints and experiences.
- Give and receive feedback with care.
- Focus criticism on ideas and code, not people.
- Assume good intent when possible.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks.
- Discriminatory language or behavior.
- Public or private abuse.
- Publishing others' private information without permission.
- Trolling, sustained disruption, or spam.

## Enforcement

Project maintainers may remove comments, issues, pull requests, or other contributions that do not follow this Code of Conduct. Repeated or severe violations may result in a temporary or permanent ban from participating in the project.

## Reporting

Please report concerns privately by contacting the repository owner through GitHub. Reports will be reviewed with care and handled as promptly as possible.
