# Support

This is a personal website project. Support is provided on a best-effort basis.

## Questions

For project setup questions, open a GitHub Discussion if available, or open an issue with the `question` label.

## Bugs

For reproducible bugs, open an issue and include:

- The affected page or feature.
- Steps to reproduce.
- Expected behavior.
- Actual behavior.
- Browser and device details when relevant.

## Security

For security issues, follow the instructions in `SECURITY.md`.
